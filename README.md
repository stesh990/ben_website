# Ben's Gym - Modern Fitness & Bodybuilding Website

A modern, attractive, and fully responsive website for Ben's Gym with a focus on fitness and bodybuilding.

## 📁 Files Included

1. **index.html** - Main HTML file with complete website structure
2. **styles.css** - Modern CSS styling with attractive colors and animations
3. **script.js** - Interactive JavaScript functionality
4. **README.md** - This file

## 🚀 How to Use

1. Open `index.html` in your web browser
2. Navigate through the website using the menu
3. Click on membership plans to see pricing
4. Use the contact form to send messages
5. Fully responsive on mobile, tablet, and desktop

## 💻 Customization

### To modify colors:
Open `styles.css` and edit the CSS variables at the top:
```css
:root {
    --primary-dark: #0a0e27;
    --primary-blue: #00d4ff;
    --accent-red: #ff0052;
    --accent-gold: #ffd700;
    ...
}
```

### To change content:
Edit `index.html` to update:
- Gym name and information
- Contact details
- Trainer names and bios
- Membership prices and features
- Program descriptions
